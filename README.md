## About This Task

This program manages the shop in the form of api
Its features include the following :

- Get Products
- Get Orders By Filter
- Create & Update Carts
- Create & Update Orders
- Role & Permission
- Check Policy
- Authentication
- Use Queue & Schedule
- Repository Pattern
- Solid & Clean Code
- RestFulApi
- ...
